# Children's world - Toy store application

-Нови технологии:
- Spring Stripe:
Оваа технологија ја користев за да овозможам наплата, односно го превзедов апито од Spring Stripe и креирав приватен и јавен клуч кој што ми овозможуваат да 
ја користат оваа технологија и нејзините функционалности, исто така ги искористив и javaScript кодот кој што ми ја овозможи оваа функционалност како и 
нејзините функционалности.
- Highcharts:
Оваа е технологија ја искористив за да прикажам статистики за сите играчки во мојата апликација.
Односно ова е javaScript библиотека за изградба на графици и пити.
Па така напишав јава скрипт код со што ги прикажувам во форма на пита, а и во форма на график статистики како процент на застапеност
на цените, како и процентот на количините на застапеност на секоја играчка
